أسئلة الحروف

الشبكة 5×5 تحتاج إلى 25 سؤالاً بالضبط. 0 / 25

ملء سريع — بدلاً من الكتابة اليدوية 25 مرة

اختر طريقة واحدة واضغط على التبويب المناسب:

قالب جاهز — مناسب للبداية

  1. اختر قالب من القائمة.
  2. اضغط تطبيق القالب — ستُملأ الحقول الـ 25 تلقائياً.
  3. بعد التطبيق، عدّل أي سؤال أو إجابة حسب رغبتك.

لصق JSON — إذا عندك البيانات جاهزة

  1. انسخ JSON يحتوي على 25 عنصر بالضبط (الصيغة أسفله).
  2. الصق في المربع.
  3. اضغط تطبيق.
[ {"letter":"ا","question":"حيوان يبدأ بحرف ا","answer":"أسد"}, {"letter":"ب","question":"فاكهة تبدأ بحرف ب","answer":"برتقال"}, ... (25 عنصر إجمالاً) ]

رفع ملف JSON — إذا عندك ملف جاهز

  1. جهّز ملف بأي اسم وينتهي بـ .json ويحتوي 25 عنصر.
  2. اختر الملف من الزر أسفله.
  3. اضغط استيراد الملف.
الصيغة المقبولة: [ {...}, {...}, ... ] ← array مباشر { "questions": [ {...} ] } ← object يحتوي questions

أزرار التسريع — داخل كل سؤال

بعد إضافة الأسئلة يدوياً، تجد في كل سؤال 3 أزرار:

  • نسخ من السؤال السابق — يملأ الحقول من السؤال اللي فوقه.
  • تكرار هذا السؤال — يضيف سؤال جديد بنفس البيانات.
  • تفريغ الحقول — يمسح بيانات هذا السؤال فقط.

سؤال 1

مجتمع عالمي مدعوم من Google Developers يهدف إلى تمكين الطلاب من خلال المعرفة التقنية وبناء الحلول.
